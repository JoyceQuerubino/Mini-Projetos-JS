KEYBOARD

O que é a Keyboard?

Os eventos do keyboard servem para verificar quando uma tecla é pressionada, desta forma é possível adicionar determinadas ações pelo teclado.

Como usar?


                    function handleKeyboard(event) {
                        if (event.key === "a") document.body.style.background = "#e8e15d";
                        else document.body.style.background = " #6abae2";
                    }
                          
                    window.addEventListener("keydown", handleKeyboard);

Pressione uma tecla e veja as alterações:

A

L

V

R
