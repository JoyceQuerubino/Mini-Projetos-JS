KEYBOARD

O que é a Keyboard?

Os eventos do keyboard servem para verificar quando uma tecla é pressionada, desta forma é possível adicionar determinadas ações pelo teclado.

Como usar?

No exemplo abaixo, criei uma função chamada ‘handleKeyboard’ que utiliza um parâmetro denominado ‘event’ (poderia ser qualquer nome), ele irá retornar várias propriedades e métodos deste evento. Uma delas é o ‘Key’, que irá verificar através do ‘if’ se a tecla 'a' foi pressionada e mudar a cor do fundo para o valor passado. 


function handleKeyboard(event) {
    if (event.key === "a") document.body.style.background = "#e8e15d";
    else document.body.style.background = " #6abae2";
    }
                          
window.addEventListener("keydown", handleKeyboard);
